This is a minor release of BDR 4 which includes new features as well
as fixes for issues identified in previous versions.


| Type | Category | Description |
| ---- | -------- | ----------- |
| Feature | User Experience | ... |
| Bug fix | User Experience | ... |
| Bug fix | Reliability and operability | ... |
| Bug fix | Reliability and operability | Prevent DDL replication failure when a non-superuser with database `CREATE` permission attempts to create a trusted extension. Previously the operation succeeded on the originating node, but failed on all other nodes with an `insufficient privileges` error. |

### Upgrades

This release supports upgrading from the following versions of BDR:

-   4.0.0 and higher
-   3.7.15 and higher

Please make sure you read and understand the process and limitations described
in the [Upgrade Guide](/pgd/latest/upgrades/) before upgrading.
